# 法律知识查询 Skill（演示）

## When to Use

- 用户希望了解**一般性**法律常识或概念（演示库为模拟数据）。
- 与劳动、婚姻、民事时效、继承等相关的常识性问题。

## Instructions

1. 提取用户问题中的核心关键词或整句作为检索用语。
2. 调用工具 **`legal_knowledge_query`**，传入参数 `query`（与用户问题一致或略作概括均可）。
3. 根据工具返回的文本，用清晰中文向用户说明，并**明确告知**内容为演示级模拟，不构成法律意见。
4. 若用户需要个案代理、文书或诉讼策略，应建议其咨询执业律师。

## Limitations

- 本技能依赖模拟知识库，**不能**替代律师或官方条文；重大决策请以正式法律文本与专业意见为准。
